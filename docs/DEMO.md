# Aftercall — Demo Script (90s max)

The demo is a **performance**, not a code walkthrough. **Hard cap: 1:30.** It
opens ON the wow (no slow build), hits two hero moments, and bakes the
"why-it's-good" into the narration instead of spending beats on it. Money/Stripe
is **cut**. Every live moment has a simulated fallback on the same timeline —
the narrator never says "real" until a phone actually rings, so a live failure
is invisible.

## Two hero moments (if a judge remembers nothing else)

1. **The cascade** — operator draws a polygon over south Tel Aviv, hits *Fire*,
   and **hundreds of grey dots cascade to green** with counters spinning up.
   "The first hour, at scale," visible in 3 seconds.
2. **The red call** — one dot turns 🔴, operator clicks it, and **a real phone on
   the table rings** (the emergency-contact escalation call, live). "The agent
   never treats — it connects a human," made physical.

## Staging checklist

- Pre-load polygon coordinates — the "draw" is muscle memory, not a fumble.
- Plant the red case — know which seeded person goes 🔴 and when. Escalation
  phone in hand, ringer ON + MAX; everything else silent-but-vibrate.
- Test room PA / Bluetooth speaker for the one live Hebrew exchange — a muffled
  call kills the "it's real" beat.
- Fallback for every live moment: the simulated dot flips on the same timeline;
  narrator never says "real" until the phone actually rings, so a failure is
  invisible. Pre-record the ring clip as backup if the network dies.
- Roles: **operator** (map only), **narrator** (never touches the laptop),
  **phone handler** (answers live call + holds escalation phone). No overlap.
- Two full rehearsals with phones actually ringing, hard-stopped at 3:00.

## Axis 1 (who pays) — backup slide, Q&A only

No stage time. One slide ready if a judge asks:
- **Readiness subscription** — cities/NGOs/HMOs/employers pay an annual retainer
  to keep Aftercall pre-loaded with their opted-in population + playbooks. A
  fire-drill contract: insurance for the hour they can't staff a phone tree.
- **Metered usage** — per person reached during an activation (Dial minutes + SMS
  pass-through + margin).
- **Why it closes** — crisis outreach is already a funded line item (Home Front
  Command, municipalities, MDA/Red Cross, employer duty-of-care). We replace the
  volunteer phone tree they already budget for — same job, infinite scale.
